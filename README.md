# applemusic

### 基于VUE仿移动端appleMusic，网易云接口，已实现大部分基本功能，（登录，搜索，歌单页，艺人页，专辑页，播放页，MV页，歌曲点赞，歌曲评论页，评论，评论点赞，回复评论，个性化推荐等）

### 根据移动端applemusic 主要分为 资料库，为你推荐，浏览，MV 四个部分，目前已完成大部分功能。 

### 技术栈： VUE全家桶，better-scroll ,awesome-swiper, js-cookie,axios

### 后端接口 ：
